# People in Space

A simple web application that demonstrates different asynchronous JavaScript patterns for fetching and displaying data about people currently in space.

## Features

- Fetches real-time data from the Open Notify API
- Displays the number of people in space
- Shows details of each person (name and spacecraft)
- Demonstrates various async programming techniques

## Files

- `index.html` - Main HTML structure
- `style.css` - Basic styling
- `async-await.js` - Implementation using async/await
- `promise.js` - Implementation using Promises
- `promises.js` - Alternative Promise implementation
- `callback.js` - Implementation using callbacks
- `callback2.js` - Another callback example
- `fetch.js` - Direct fetch API usage

## How to Run

1. Clone the repository
2. Open `index.html` in a web browser
3. Click the "View all the People" button to load data

## API

Uses the [Open Notify API](https://raw.githubusercontent.com/openai/gpt-demo-data/main/astronauts.json) for astronaut data.
